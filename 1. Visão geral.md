# Sobre o minicurso
## 1. Pré-requisitos
- JavaScript básico
- Conhecimentos introdutórios de Node.js
- Familiaridade com React
- Uso básico de terminal e npm

## 2. Carga horária
| Atividade                                  | Carga Horária |
| ------------------------------------------ | ------------- |
| Aulas gravadas                             | ~1.5h            |
| Follow along e implementação guiada        | ~2h            |
| Desenvolvimento da atividade prática final | ~3h            |

Total: 6h

## 3. Objetivos do minicurso
- Compreender a aplicação prática de testes unitários, de integração e E2E
- Capacitar o espectador a
	- criar testes
	- rodar testes
	- mockar dependências
	- validar fluxo principal


## 4. Como utilizar os nossos vídeos
- Será disponibilizado um projeto full-stack (React + Vite + Express) que utilizaremos como base para o desenvolvimento de testes.
- Esperamos que o espectador utilize esse projeto para acompanhar o instrutor durante o decorrer das aulas, realizadas de forma expositiva.
- Todos os testes escritos durante as aulas serão disponibilizados apenas após o início do período de aplicação do desafio final.

## 5. Desafio final
- Ao final do minicurso, os participantes deverão desenvolver uma atividade prática avaliativa baseada na implementação de novos testes automatizados para o nosso projeto base.
- Esse desafio será o critério avaliativo utilizado para a obtenção do certificado.

## 6. Projeto base
Utilizaremos um **sistema pequeno mas com fluxo completo** (full-stack), com as seguintes funcionalidades:
- login
- cadastro
- CRUD simples
- formulário
- persistência fake

Com esse sistema, exploraremos as seguintes atividades:

- **Vitest**:
  - validação
  - services
  - controllers
- **Cypress**:
  - login
  - fluxo do usuário
  - integração frontend/backend
